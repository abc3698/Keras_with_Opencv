# Keras_with_Opencv
케라스로 학습한 모델을 Opencv DNN을 통해 불려오는 예제

# Prerequisite
* Tensorflow-gpu 1.13.1
* Opencv 4.1
* Visual Studio 2017

# 실행순서
1. python keras_mnist_cnn.py 실행하여 my_model.pb 생성
2. source.cpp를 visual studio 2017에 Import
3. Opencv 4.0 라이브러리를 Include 및 lib 포함
4. 실행

# 프로그램 사용방법
1. 숫자를 그린 후 스페이스 클릭
